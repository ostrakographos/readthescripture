## Why NoDerivatives, for a repository about transparency

Two reasons, stated plainly:

1. **Scripture licensing.** The site quotes the New Revised Standard Version Updated Edition under permission terms that require the quoted text to be unaltered. A license permitting derivative works would purport to grant modification rights over quoted text this project does not own and cannot sublicense.
2. **Provenance.** Every grade in `grading/grades.jsonl` is bound to an exact version of the file it scored. A modified copy of a page carrying the original's grade block would misrepresent the record. This repository exists to *be* a record; records are shared verbatim.
